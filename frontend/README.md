## Getting Started On Frontend

1. Install bun:

```bash
powershell -c "irm bun.sh/install.ps1 | iex"
```

2. Install dependencies:

```bash
cd frontend
bun i
```

3. Run the development server:

```bash
bun dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
